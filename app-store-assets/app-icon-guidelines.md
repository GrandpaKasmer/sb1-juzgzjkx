# TerraTime Grounds - App Icon Guidelines

## App Icon Requirements

### iOS App Store Icon
- 1024x1024 pixels
- PNG format
- No alpha channel/transparency
- RGB color space
- Fully opaque
- No rounded corners (Apple will add rounding)

### iOS Device Icons
- 180x180 pixels (iPhone)
- 167x167 pixels (iPad Pro)
- 152x152 pixels (iPad, iPad mini)
- 120x120 pixels (iPhone)
- 76x76 pixels (iPad)

### Android App Icon
- 512x512 pixels (Play Store)
- 192x192 pixels (launcher icon)
- 144x144 pixels (launcher icon)
- 96x96 pixels (launcher icon)
- 72x72 pixels (launcher icon)
- 48x48 pixels (launcher icon)
- 36x36 pixels (launcher icon)

## Design Guidelines
- Use the TerraTime logo with the green color scheme
- Maintain simplicity and recognizability at small sizes
- Avoid text in the icon
- Use the established brand colors:
  - Primary Green: #22c55e
  - Secondary Blue: #3b82f6
  - Background: White or light gray
- Ensure the icon is recognizable at a glance
